## ‚ú® Features

### Monitor Testing
- **Dead Pixel Detection** - Identify stuck or dead pixels
- **Refresh Rate Testing** - Verify monitor refresh rates with detailed frame time graphs
- **Ghosting & Motion Blur Tests** - Assess display response times with UFO test
- **Color Uniformity Analysis** - Check for color consistency
- **Gradient & Pattern Tests** - Evaluate color reproduction
- **Text Clarity Assessment** - Test font rendering quality
- **Burn-in Prevention Tools** - Pixel exercising utilities
- **Monitor Information** - Display specifications and capabilities
- **Audio Testing** - Basic audio output verification

### Gamepad Testing
- **Automatic Gamepad Detection** - Detects and displays all connected controllers
- **Button Testing** - Real-time visual feedback for all button presses
- **Analog Stick Visualization** - Live analog stick position with circularity testing
- **Trigger Pressure Display** - Visualize L2/R2 and LT/RT trigger pressure
- **Vibration Testing** - Test controller vibration with manual controls and custom patterns
- **Pattern Editor** - Create, save, and reorder custom vibration patterns
- **Xbox & PlayStation Support** - Icons and layouts for both controller types
- **D-Pad Testing** - Dedicated directional pad visualization

## üöÄ Quick Start

1. Download the latest release for your platform
2. Install and run CMS-Toolkit
3. Select a test from the main menu:
   - Choose from various monitor tests
   - Click "Gamepad Tester" for controller diagnostics
4. Use F11 to toggle fullscreen mode during monitor tests
5. For gamepad testing, connect your controller and follow on-screen instructions

## üéÆ Gamepad Testing Guide

1. Connect your gamepad via USB or Bluetooth
2. Open the Gamepad Tester from the main menu
3. Your controller will be automatically detected
4. Test all buttons, sticks, triggers, and D-pad
5. Use vibration tests to verify motor functionality
6. Create custom vibration patterns for advanced testing
